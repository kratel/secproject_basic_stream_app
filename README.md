# Ubuntu setup

```bash
sudo apt-get install scrot
```
For system python2
```
sudo apt-get install python-tk python-dev
```

For system python3
```
sudo apt-get install python3-tk python3-dev
```

_Note: for specific version of python3.x
```
sudo apt-get install python3.x-tk python3.x-dev
```
